# Bilibili Audio Player Chrome Extension - 使用说明

## 功能介绍

Bilibili Audio Player 是一个 Chrome 扩展，可以从 Bilibili 视频中提取音频并独立播放。主要功能包括：

1. 从 Bilibili 视频 URL 或 BV 号中提取音频，**并采用按需获取机制，避免链接过期问题**。
2. 独立播放音频，关闭 Bilibili 视频页面不影响音频播放。
3. **播放器窗口复用**：优先使用已打开的播放器窗口，避免重复打开。
4. 仅在提取音频时，由后台按白名单临时读取必要的 Bilibili Cookie；不会保存或同步登录凭据。
5. 自动检测当前页面是否为 Bilibili 视频页面。
6. 提供浮动按钮、弹出窗口和独立播放器窗口。
7. **播放历史：** 自动记录最近播放的音频（使用视频 ID 存储，确保持久有效），可在扩展弹出窗口中查看、播放历史记录，并支持在设置页面查看完整历史和清空。
8. **自定义播放合集：** 用户可以创建和管理自己的音频收藏。
    * 可在设置页面创建、重命名、删除播放合集。
    * **在设置页面可直接向指定合集内粘贴 B 站视频链接进行添加**。
    * 在播放器中可将当前音频添加到指定的播放合集。
    * 可在设置页面查看播放合集内容，并播放其中的音频。
9. **分P选集：** 自动识别视频的全部分P，支持 `?p=` 链接、播放器内切换选集和播完自动下一P。
10. **播放倍速：** 支持 0.25×–3× 播放速度、±0.05× 微调，并自动记住上次选择。

## 安装说明

### 开发者模式安装

1. 克隆或下载本项目，运行 `npm run build` 生成 `dist/` 目录
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目的 `dist/` 目录

### 从 Chrome 网上应用店安装（未发布）

暂未上架，请使用开发者模式安装。

## 使用方法

### 方法一：在 Bilibili 视频页面使用

1. 访问任意 Bilibili 视频页面
2. 点击页面右下角出现的"提取音频"浮动按钮
3. 音频将在新窗口中打开并自动播放

### 方法二：通过扩展图标使用

1. 点击 Chrome 工具栏中的扩展图标
2. 如果当前页面是 Bilibili 视频页面，将显示视频信息和"播放此视频音频"按钮
3. 或者手动输入 Bilibili 视频链接或 BV 号（如 `BV1xx411c7mD`），点击"提取并播放音频"

### 登录视频

需要登录的视频会使用当前浏览器中的 Bilibili 登录状态。扩展不会要求复制 `SESSDATA`，也不会将登录 Cookie 写入扩展存储；请直接在 Bilibili 网站登录。

## 播放器使用

播放器窗口提供以下功能：
- 播放/暂停控制
- 空格键播放/暂停
- 上一集/下一集（播放合集模式下为上一首/下一首）
- 快退 10 秒/快进 10 秒
- 进度条拖动
- 音量调节
- 分P选集（多P视频）
- 0.25×–3× 播放速度与 ±0.05× 微调
- 添加到播放合集按钮（将当前音频添加到用户创建的播放合集）
- 设置按钮（快速访问设置页面）

## 注意事项

- 本扩展仅用于个人学习和使用
- 请尊重 Bilibili 的版权和使用条款
- 某些视频可能需要登录才能提取音频，请确保在浏览器中已登录 Bilibili
- 如遇到问题，请尝试重新登录 Bilibili 或检查网络连接
